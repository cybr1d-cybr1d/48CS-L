# 48CS-L
A going away gift I made for a friend. It uses 2 ATMega 2560s to drive almost 100 LEDs. There are 10 modes for the LAN cable and 14 modes for the binary. The top chip controls the LAN cable LEDs and the bottom chip controls the binary LEDs and the afterburners on the jet. LEDs for the LAN cable are 1206 smd soldered upside down and all others are right angle 1206 SMD LEDs.
